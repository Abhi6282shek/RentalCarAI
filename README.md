# RentalCarAI

A full-stack AI-powered Car Rental Management System.

## Features

- User Authentication
- Car Listing
- Car Details
- Booking System
- My Bookings
- AI Recommendations
- Search Filter
- Brand Filter
- Fuel Filter
- Price Filter
- Responsive Design

## Tech Stack

Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Spring Boot
- Java 21

Database:
- MySQL
